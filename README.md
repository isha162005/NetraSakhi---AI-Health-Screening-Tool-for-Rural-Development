# NetraSakhi---AI-Health-Screening-Tool-for-Rural-Development
AI powered tool using image processing methods like Hough Transform, Canny Edge Detection and Contour method along with CNN for disease and deficiency detection. Classification of Anaemia, Chickenpox, Ringworm and Leukonychia (Zinc Deficiency) along with Vaccine / Polio drops date prediction via App for rural development.
